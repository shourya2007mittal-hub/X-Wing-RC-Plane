# X-Wing-RC-Plane

I made a Tailless X-Wing Quad-Motor RC Plane inspired by LEGO with a custom flight controller based on STM32. It is based on STM32 and end design is  inspired by LEGO.

<img width="702" height="625" alt="Screenshot 2026-06-18 231855" src="https://github.com/user-attachments/assets/20c4ea48-d89a-41d5-b5eb-a650b83a4d69" />

## Features

Microcontroller: STM32F401CBUx<br>
IMU Sensor: MPU-6050 , ICM-20948<br>
Barometer: BMP280<br>
Design inspired by LEGO<br
